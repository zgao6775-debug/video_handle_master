特效自动添加

- 自动识别适合添加特效的场景
- 按场景自动应用转场淡入淡出与滤镜
- 通过帧差与颜色直方图检测分镜

使用

1. 安装依赖：pip install moviepy opencv-python numpy
2. 运行示例：
   python video-effects-add/auto_effects.py --input d:/rubbish/video-handle/video-Motion-Analysis/example.mp4 --output d:/rubbish/video-handle/video-effects-add/output_with_effects.mp4
