GUI_code

说明
- 提供一个可交互GUI，集成：关键帧+摘要统一输出、自动特效、动作分析、目标检测、自动配乐、字幕烧录
- GUI 运行产生的所有输出会保存到仓库根目录的 output/RunID/ 下

运行
python GUI_code/app.py

常用设置说明
- 翻译/多语言：选择是否生成英文字幕/摘要；更多语言（如日文/法文）需要勾选“联网增强”
- 识别精度：标准（推荐）更快；更精确（更慢）适合口音/噪声较多的视频
- 生成字幕文件：输出 .srt 文件，供烧录到视频或外部播放器加载
- 自动配乐：支持三种方式（本地音频合成/提示词生成/自动生成），生成 video_with_music.mp4

依赖
- Python 自带 tkinter（一般已自带）
- ffmpeg（字幕烧录/自动剪辑分割等需要）
- 已安装的项目依赖：opencv-python、moviepy、mediapipe、whisper/torch（按功能启用）
